# StyleME Beta (Archived)

**Previous Work – StyleME Beta (formerly StyleAI)**

We previously built an early beta of StyleME (originally called StyleAI). While it demonstrated promising core ideas, it lacked several features we are now adding to the final product.

In the spirit of transparency and open learning, we are releasing this beta publicly on OpenZeg as-is for educational purposes and community inspiration.

---

## About This Repository

This is an archived beta version of our AI fashion intelligence tool. The current StyleME is being rebuilt with a stronger focus on privacy, honesty, and amplifying personal style.
